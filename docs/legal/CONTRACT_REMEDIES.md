**Step 1: Request Modification (if order still open)**
- Navigate to order page
- Click "Request Modification"
- Specify exactly what was not delivered

**Step 2: File Formal Dispute**
- Access: Orders â†’ Resolution Center

## IV. LEGAL REMEDIES OUTSIDE FIVERR

### A. If Fiverr Dispute Fails

**Small Claims Court:**
- Jurisdiction: Contractor's location or your location (varies by state)
- Limit: Typically $5,000-$10,000 (check local limits)
- Process: File claim, serve contractor, present evidence at hearing

**Arbitration:**
- Check if Fiverr ToS requires arbitration
- May be binding for platform-based disputes
- Cost varies; may not be practical for small amounts

### B. Contract-Specific Provisions

Review your Master Services Agreement for:

- [ ] Dispute resolution clause (arbitration/mediation required?)
- [ ] Governing law/jurisdiction
- [ ] Termination provisions
- [ ] Warranty provisions
- [ ] Limitation of liability clauses
- [ ] Notice requirements for breach claims

## VII. DECISION MATRIX

### Option 1: Pursue Full Refund Only
**When to choose:** Amount is significant; don't want to work with contractor again
- File Fiverr dispute
- Seek complete refund
- Engage replacement developer
- Use existing codebase as foundation

### Option 2: Demand Contract Completion
**When to choose:** Contractor shows willingness; prefer continuity
- Issue formal cure notice (72-hour response deadline)
- Require milestone checkpoints with holdback
- Demand code repository access
- Proceed only with verified deliverables

### Option 3: Hybrid Approach
**When to choose:** Partial refund acceptable; want to preserve some relationship
- Negotiate partial refund for work not done
- Accept reduced scope for remaining work
- Establish strict oversight for remaining milestones
